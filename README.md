# Generator microlib

A Yeoman generator for browser microlibs.  

It includes:

* README.md
* Grunfile
* bower.json
* package.json
* test scaffolding (optional)

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator **locally**: `npm install generator-microlib`
- Run: `yo microlib`
